# 月考核作业：匿名场景月度文档生成（v2 - OfficeCLI 版）

用于学前教育场景月度考核文档的完整生成工作流，基于 [`officecli`](https://officecli.ai) 单二进制 CLI。

## 技能概述

本技能用于自动化处理匿名教育场景中的月度材料：
- 查找共享盘中的考核文件
- 生成或编辑游戏观察记录表
- 生成或编辑运动观察记录表
- 生成或编辑备课文档

## 触发词

## 三大文档模板的 OfficeCLI 命令流

### 模板 1：游戏观察记录表（6 行 4 列）

```bash
officecli create game.docx
officecli add game.docx /body --type table --prop rows=6 --prop cols=4

officecli set game.docx '/body/tbl[1]/tr[1]/tc[1]' --prop text="观察时间" --prop bold=true
officecli set game.docx '/body/tbl[1]/tr[1]/tc[2]' --prop text="2026年6月"
officecli set game.docx '/body/tbl[1]/tr[1]/tc[3]' --prop text="观察地点" --prop bold=true
officecli set game.docx '/body/tbl[1]/tr[1]/tc[4]' --prop text="活动区A"
```

### 模板 2：运动观察记录表（5 行 6 列）

最高频操作：追加新月份。

```bash
./scripts/append_motion_table.sh motion.docx 4 2026年6月
```

完整填充流程：

```bash
officecli add motion.docx /body --type table --from /body/tbl[4]
officecli set motion.docx '/body/tbl[5]/tr[1]/tc[2]' --prop text="2026年6月"
python3 scripts/gen_batch.py motion.docx 5 /tmp/jun_content.json \
  | officecli batch motion.docx --json
```

`/tmp/jun_content.json` 示例：

```json
{
  "R0": {"C2": "2026年6月", "C4": "教师A", "C6": "幼儿A"},
```

## 内容规范

| 格式项 | 参数值 | officecli prop |
|---|---|---|
| 字体 | 宋体 | `font.latin=宋体 font.ea=宋体` |
| 字号 | 12pt | `size=12pt` |
| 行距 | 1.5 倍 | `lineSpacing=1.5x` |
| 内容列首行缩进 | 2 字符 | `firstLineChars=200` |

## 完整工作流示例

```bash
python3 -c "
import os
src = '/Volumes/shared/.../运动观察记录.docx'
fd = os.open(src, os.O_RDONLY | os.O_NONBLOCK)
data = b''
while True:
    chunk = os.read(fd, 65536)
    if not chunk:
        break
    data += chunk
os.close(fd)
with open('/tmp/motion_work.docx', 'wb') as f:
    f.write(data)
"

python3 scripts/gen_batch.py /tmp/motion_work.docx 5 /tmp/jun_content.json \
  | officecli batch /tmp/motion_work.docx --json

cp /tmp/motion_work.docx "/Volumes/shared/.../运动观察记录.docx"
```
