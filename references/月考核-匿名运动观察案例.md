## 写作规范

### 观察记录

- 写具体场景，不写空泛总结
- 保留时间线
- 可以使用匿名直接引语
- 适当记录行为数据

### 分析与评价

建议采用三段式或四段式：
1. 总体判断
2. 能力发展
3. 情绪或社会性表现
4. 后续观察方向

### 支持策略

建议从以下维度组织：
1. 难度递进
2. 材料变化
3. 游戏化情境
4. 同伴学习
5. 自我评估与安全意识

## 技术要点

### 大文件读取

```python
import os

fd = os.open('/Volumes/shared/.../运动观察记录.docx', os.O_RDONLY | os.O_NONBLOCK)
data = b''
while True:
    chunk = os.read(fd, 65536)
    if not chunk:
        break
    data += chunk
os.close(fd)
```

### 扫描共享盘时避免卡死

```python
import os
import signal

def handler(signum, frame):
    raise TimeoutError

signal.signal(signal.SIGALRM, handler)
signal.alarm(25)
try:
    for dirpath, dirnames, filenames in os.walk('/Volumes/shared', topdown=True):
        pass
except TimeoutError:
    pass
finally:
    signal.alarm(0)
```
